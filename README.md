# 日報変換ツール

効率的な日報入力と月報生成を支援するCLIツール

## 概要

このツールは日々の業務報告（日報）の入力を効率化し、蓄積されたデータから任意のフォーマットの月報を生成するための支援ツールです。

### 主な機能

- 指定されたフォーマットに従った日報入力支援
- 日報データの管理（保存、閲覧、編集、検索）
- 生成AIと連携した柔軟な月報生成
